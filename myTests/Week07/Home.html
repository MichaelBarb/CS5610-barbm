Summary

Experiment 1:   Setup Full Featured Responsive Web Page

In previous experiments I had tried several things to build a good home page. I could never get a look and feel that looked professional. In particular I found js-ui very disappointing. Bootstrap was able to give me the look I was after so I am trying an experiment to see how it will work.

I started with the example from the Kindle book referenced. I set it up in my website with all the appropriate paths to css and js files. I started refactoring it for use as a new home page for this site.

Experiment 2:   Put Picture in Navbar

One of the first things I needed was to put my picture in the navbar. Putting the picture in was easy but getting it to fit properly was not. After several tries I found the solution was much easier than originally attempted.

Sizing was not that hard. I found references that said that using the attributes in image for height and width can be over ridden by css styles. They recommend putting these settings in “style” so they cannot be overridden.

For proper positioning I found default padding was preventing it from being properly placed. I had to reset the padding-top to zero.

The following code snippet is the final result:

  • <a class="navbar-brand" href="#" style="padding-top:0">
  •     <img src="../../myImage/myPicture.jpg" alt="" style="width:42px;height:51px" />
  • </a>

In the process I learned a lot about Bootstrap navbar and the mny parts of it. The next experiment also involves the navbar.

Experiment 3:   Put Metric links in Pull-Down Menu

There are many metrics and utilities that are to be available from the home page. I put these in pull downs in the navbar. I found using Bootstrap this was a fairly easy task.

Experiment 4:   Edited Carousel

One thing I not like about it is the change in height from image to image. You can fix the height but then you lose its responsive nature. I tried several things but could not get it behave properly. Some help on the web suggested you need to do an override in JavaScript. I will tackle this later but by itself it will be a major investment of time.

Experiment 5:   Add Major Links

Using Bootstraps grid I set up 3 cells for the other major links in this site. They are Experiments, Blog, and Project.

I put an icon in the link / title of each. This system use a custom font and is much better than the icon system in js-ui. The fonts are vector drawings and scale nicely. In js-ui is has a single large png file with all icons and then masks all but the desired one. Since they are bitmaps they do not scale well and are difficult to change colors.

Also doing each one of the major links as a cell allowed me to put a short explanation below each link.

Below the 3 cells of the major links I also but in a 2 cell row for a link to Github. I could only get this icon as bit map and its color does not match well. I do not have the tools to manipulate the color of a svg file.
