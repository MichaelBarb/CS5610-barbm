In the first experiments the JavaScript was embedded in the html. This experiment moves it to a separate file.

Note that this script sets up the buttons in order. Formatting is not tied uniquely to each button. The next experiment will provide unique button definition.

In Visual Studio there is a common convention for naming the code behind file, in this case it is the ".js" file. You start with the full name of the view and add the required extension.

In the case of this experiment the view is jQuery2.html. The code behind would be jQuery2.html.js.

  • <link href="../../myCSS/myTheme.css" rel="stylesheet" />
  • <link href="../../Scripts/jquery-ui-1.11.3/jquery-ui.min.css" rel="stylesheet" />
  • <script src="../../Scripts/jquery-1.11.2.min.js"></script>
  • <script src="../../Scripts/jquery-ui-1.11.3/jquery-ui.min.js"></script>
  • <!--<link href="../../Scripts/jquery-ui-1.11.3/jquery-ui.theme.css" rel="stylesheet" />-->
  • <script src="jQuery2.html.js"></script>
