I did not like the way that Bootstrap changed the defaults. I created my own theme in a .css file. I made the following changes:

  1. Loss of default margins and padding. I moved it from the page code and but the default in myCCS.
  2. Color of the text to Black.
  3. Color of the headers to DarkBlue.

In the process I learned better how to use the PageInspector in Visual Studio

To see the code developed for thi experiment see the source for entire myTheme.css file.
