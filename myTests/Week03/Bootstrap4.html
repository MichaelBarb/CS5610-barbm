Style Reset: Reset Before Bootstrap

Summary

In class they mentioned using a style reset to get rid of all the defaults from the browser. I used one to see how many things the browser is changing.

It also shows how many things that Bootstrap and what is the combination of the browser and Bootstrap. I created the following 3 pages/experiments:


  1. Reset Only.
  2. Reset before Bootstrap.
  3. Reset after Bootstrap.
  4. For Reference Bootstrap and browser defaults.

In this experiment the lines in the header are as follows:

  • <link href="../../myCSS/html5reset-1.6.1.css" rel="stylesheet" />
  • <link href="../../Content/bootstrap/bootstrap.min.css" rel="stylesheet" />
  • <link href="../../myCSS/myTheme.css" rel="stylesheet" />
