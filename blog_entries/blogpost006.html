One thing I like about this is that it install Bootstrap into the project with source and documentation. This will allow more experimentation with Bootstrap and to be better able to understand what it does.

A complication of this is that it uses the .less files that need to be compiled. It took me awhile to figure out I needed to install the Web Essential extention into Visual Studio. This extension contains a lot of important aids for Visual Studio that all should have.

  • 1. Links to source code, i.e. File utility links are broken.
  • 2. The documentation and source code should be in single file. In your Master Page experiment, I can see 5 entries for in story file which is not a proper way of documenting an experiment.

Regarding the second requirement: In several of the other TA's websites they put the sorce code references in the story. For this experiment I added them to the experiment page and left them in the story.
